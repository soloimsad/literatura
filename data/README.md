# Data

Carpeta para datasets locales, datos preparados e imagenes de prueba.

- `dental-disease-panoramic-detection-dataset/`: dataset de hallazgos dentales en formato YOLO/COCO.
- `teeth_segmentation/`: dataset de segmentacion dental en formato Supervisely.
- `_prepared_teeth_yolo_seg/`: conversion local a YOLO-seg generada por `src/dental_xray_pipeline.py`.
- `samples/`: imagenes pequenas para pruebas de inferencia.

Los datasets pesados se mantienen fuera de git.
