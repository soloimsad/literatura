# Notebooks locales

Flujo recomendado para la presentacion del proyecto:

1. `01_preparacion_datos_local.ipynb`: audita y prepara datasets locales.
2. `02_inferencia_pieza_dental.ipynb`: ejecuta el modelo de pieza dental.
3. `03_inferencia_tratamiento.ipynb`: ejecuta el modelo local de posible tratamiento.
4. `04_demo_integrada_local.ipynb`: demo compacta para mostrar resultados.

Todos los notebooks asumen ejecucion local desde este equipo y usan las rutas
centralizadas en `src/model_registry.py`.
